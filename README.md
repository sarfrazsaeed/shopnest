# 👟 ShopNest — Premium Sneakers & Fashion

> A commercial-grade e-commerce frontend built with Vite + React 18 + TypeScript + GSAP + Framer Motion

[![Live Demo](https://img.shields.io/badge/Live-Demo-F5B800?style=for-the-badge)](https://sarfrazsaeed.github.io/shopnest/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.6-3178C6?style=for-the-badge&logo=typescript)](https://www.typescriptlang.org/)

## 🛍 Features
- Product grid with filters, search (debounced), sort, grid/list view
- Product detail with image gallery, color/size selector, quantity
- Cart drawer (Zustand) with quantity management
- Wishlist with move-to-cart
- 3-step checkout with shipping form + card UI
- GSAP ScrollTrigger animations + Framer Motion transitions
- Dark luxury design — Playfair Display + Inter

## 🛠 Stack
React 18 · TypeScript · Vite · Tailwind CSS · Framer Motion · GSAP · Zustand · React Router v6
